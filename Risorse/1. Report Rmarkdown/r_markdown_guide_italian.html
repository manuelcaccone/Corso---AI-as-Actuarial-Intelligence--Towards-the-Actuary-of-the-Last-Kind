R Markdown per Report Attuariali

Una guida completa per creare report attuariali professionali usando R Markdown

1. Primi Passi

Installazione di R & RStudio

Prima di iniziare con R Markdown, devi configurare il tuo ambiente di sviluppo. Segui questi passaggi per il tuo sistema operativo:

  1. Installa R: Visita CRAN e scarica l'ultima versione di R
  2. Esegui l'installer e accetta tutte le impostazioni predefinite
  3. Installa RStudio: Visita RStudio.com
  4. Scarica la versione gratuita RStudio Desktop
  5. Installa RStudio (assicurati che R sia installato prima)
  1. Installa R: Scarica R-x.x.x.pkg da CRAN
  2. Fai doppio clic sull'installer e segui la procedura guidata
  3. Installa RStudio: Scarica da RStudio.com
  4. Trascina RStudio nella cartella Applicazioni
  1. Installa R: Usa il tuo gestore di pacchetti
  2. Ubuntu/Debian: sudo apt-get install r-base
  3. Aggiorna le sorgenti dei pacchetti e installa le dipendenze
  4. Installa RStudio: Scarica .deb o .rpm da RStudio.com

Pacchetti Essenziali

Installa questi pacchetti per iniziare con R Markdown per i report attuariali:

Console R 
install.packages(c("rmarkdown", "knitr", "tinytex", "kableExtra", "dplyr", "ggplot2"))

# Installa TinyTeX per l'output PDF
tinytex::install_tinytex()

Panoramica dei Pacchetti:

  • rmarkdown: Funzionalit√† core di R Markdown
  • knitr: Generazione dinamica di report
  • tinytex: Distribuzione LaTeX leggera per output PDF
  • kableExtra: Formattazione avanzata delle tabelle
  • dplyr: Manipolazione dei dati
  • ggplot2: Visualizzazione dei dati

# Riassunto Esecutivo

Questo report presenta l'analisi attuariale per il periodo che termina il `r Sys.Date()`.

# Analisi dei Dati

```{r}
# Dati attuariali di esempio
data <- data.frame(
  Prodotto = c("Vita Temporanea", "Vita Intera", "Vita Universale"),
  Premi = c(1000000, 2000000, 1500000),
  Sinistri = c(650000, 1200000, 900000),
  Loss_Ratio = c(0.65, 0.60, 0.60)
)

kable(data, caption = "Analisi Loss Ratio") %>%
  kable_styling(bootstrap_options = c("striped", "hover"))
```

# Conclusioni

L'analisi mostra loss ratio favorevoli su tutte le linee di prodotto.

3. Intestazioni YAML per Report Attuariali

Configurazione Documenti Word

Configura l'output Word per report attuariali professionali:

YAML Documento Word 
---
title: "Report Attuariale"
author: "Il Tuo Nome"
date: "`r Sys.Date()`"
output: 
  word_document:
    reference_docx: template.docx
    toc: true
    toc_depth: 3
    number_sections: true
    fig_caption: true
    fig_width: 8
    fig_height: 6
    df_print: kable
---
